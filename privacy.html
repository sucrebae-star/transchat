1. 수집하는 정보

Transchat는 서비스 운영과 기능 제공을 위해 다음 정보를 수집하거나 처리할 수 있습니다.

  • 회원가입 및 로그인 정보: 아이디, 비밀번호, 이름, 언어 설정, 확인 질문 및 정답
  • 프로필 정보: 프로필 사진, 성별, 나이, 상태 정보 등 이용자가 직접 입력한 정보
  • 채팅 정보: 이용자가 전송한 메시지 원문, 이미지, 캡션, 채팅방 정보
  • 기기 및 앱 정보: 푸시 알림 토큰, 앱 버전, 기기 환경 정보
  • 음성 입력 기능 사용 시: 음성 인식을 위해 기기 마이크 권한을 사용할 수 있습니다

2. 개인정보 이용 목적

  • 회원 식별, 로그인 유지, 계정 복구, 보안 질문 확인
  • 채팅, 연결 요청, 이미지 전송, 수동 번역 등 핵심 기능 제공
  • 푸시 알림 제공 및 서비스 상태 동기화
  • 오류 분석, 안정성 개선, 서비스 품질 향상
  • 부정 사용 방지 및 보안 대응

3. 개인정보 보관 기간

개인정보는 서비스 제공 목적이 달성될 때까지 보관하며, 이용자가 삭제를 요청하거나 계정을 탈퇴하는 경우 관련 법령에 따라 필요한 범위를 제외하고 지체 없이 삭제하거나 익명화합니다.

테스트 운영 또는 기능 개선 과정에서 로그가 일시적으로 보관될 수 있으나, 보안 및 안정성 확인 목적 외의 용도로 사용하지 않습니다.

4. 제3자 제공 및 외부 처리

Transchat는 원칙적으로 이용자의 개인정보를 외부에 판매하거나 임의로 제공하지 않습니다. 다만, 서비스 기능 제공을 위해 아래와 같은 외부 서비스가 사용될 수 있습니다.

  • OpenAI: 이용자가 요청한 수동 번역 기능 제공
  • Firebase Cloud Messaging: 푸시 알림 전송
  • 호스팅/서버 인프라: 서비스 운영, 저장, 보안 관리

외부 서비스는 해당 기능 수행에 필요한 범위에서만 정보를 처리합니다.

5. 권한 사용 안내

  • 마이크 권한: 음성 → 텍스트 입력 기능을 사용할 때만 사용

  • 알림 권한: 새 메시지 및 초대 알림 제공 시 사용

이용자는 기기 설정에서 각 권한을 언제든지 허용하거나 철회할 수 있습니다.

6. 이용자의 권리

  • 본인의 개인정보 열람, 수정, 삭제 요청
